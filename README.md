# 🌾 Krishi Cure AI

Krishi Cure AI is an AI-powered smart agriculture web application designed to help farmers identify crop diseases and get proper treatment suggestions in multiple languages.

## 🚀 Features

* 🌱 Crop disease symptom detection
* 🤖 AI-powered disease prediction
* 📊 Risk percentage analysis
* 🧪 Organic treatment suggestions
* 💊 Scientific treatment suggestions
* 🌐 Multi-language support

  * Gujarati
  * Hindi
  * English
* 🔐 Google Authentication Login
* 🌦️ Live Weather Information
* 📍 Automatic Location Detection
* 🗺️ Manual Location Selection
* 🤖 AI Agriculture Chatbot
* 🎤 Voice-enabled Chatbot Support
* 🔊 AI Voice Assistant
* 📱 Fully Responsive Design
* ⚡ Fast and User-Friendly Interface

## 📈 How Krishi Cure AI Works

1. Sign in with Google
2. Select Crop Category
3. Select Crop
4. Select Symptoms
5. Get AI Disease Prediction
6. View Risk Analysis
7. Receive Treatment Suggestions
8. Check Weather Conditions
9. Ask Questions using AI Chatbot
10. Type or Speak using Voice Input
